## Oto

![oto](./images/oto.png)

Oto is a theme for the Zed code editor. It's a dark theme with a focus on clear colours and a clean minimal look.

### Installation

```bash
git clone (this repo)
```

Then Run:

```bash
sudo find ~/oto/ -type f \( -iname '*.json' \) -exec cp {} ~/.config/zed/themes \;
```
